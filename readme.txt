Toiminnot:

Kortin tiedot -> Valitse YKSI tai USEAMPIA kortteja. Tiedot tulostuvat tekstilaatikkoon. Jos valitset YHDEN kortin, ovien valintaruudut valitaan automaattisesti

Pääsy ovelle -> Valitse YKSI ovi, tekstikenttään tulostuu kortit ja niiden valintaruudut valitaan automaattisesti.

Lisää kulkuoikeuksia -> Valitse YKSI tai USEAMPIA kortteja ja/tai ovitunnuksia. Ovet tallentuvat korteille

Yhdistä kortteja -> Valitse KAKSI korttia, niiden kulkuoikeudet yhdistetään.

Poista kulkuoikeuksia: -> Valitse YKSI tai USEAMPIA kortteja ja/tai ovitunnuksia. Valitut kulkuoikeudet poistetaan korteilta.

Lisää kortti -> Kirjoita pyydetyt tiedot kenttiin. Uusi kortti luodaan painamalla nappia.

Poista kortti -> Valitse YKSI tai USEAMPIA kortteja, ne kaikki poistetaan.

Tiedosto -ylävalikko:
- Uusi -> Kaikki kortit poistetaan
- Avaa korttitiedosto -> Voit avata .txt -tiedoston, joka sisältää korttien
  tiedot csv -muodossa: id-tunnus;nimi;kulkuoikeudet,pilkulla,erotettuna
- Tallenna korttitiedosto -> Voit tallentaa muokkaamasi kortit uuteen
  csv -tiedostoon
- Lopeta Ohjelma -> Ohjelma sammutetaan ja tallentamattomat tiedot katoavat

Apua -ylävalikko:
- Käyttöohjeet -> Tämä käyttöohje avautuu popup -ikkunaan
